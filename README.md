# Data Jobs Dashboard w/ Power BI

![Data Jobs Dashboard Page 1](/images/Data_P1.png) 


## Introduction

This dashboard was developed for **professionals exploring or switching into data-related roles** who need a clearer picture of the job market.
Using a real-world dataset of data job postings covering roles, salaries, and locations this consolidates everything into a single interactive dashboard

### Dashboard File
You can find the file for the dashboard here: [`Data_Jobs_Dashboard.pbix`](Data_Jobs_Dashboard.pbix).  

## Skills Showcased

This project was a journey through key Power BI features. Here's a look at what we mastered:
*   **Implicit Measures:** Key indicators such as `Median Yearly Salary` and `Job Count` were derived directly within Power BI to support insight generation.
*   **Core Visualisations:** **Column, Bar, Line, and Area charts** were used to compare job volumes and observe changes across different roles.
*   **Geographic Insights:** **Map visuals** were applied to highlight the global spread of data job opportunities.
*   **KPI & Detailed Tables:** **Card visuals** summarize important metrics while tables allow users to explore job-level details.
*   **Report Layout & Visual Design:** The dashboard layout focuses on clarity and usability, combining both standard and advanced chart types for effective storytelling.
*   **Interacivity & Navigation:**
    -   Slicers enable filtering by job role.
    -   Buttons and bookmarks create smooth page navigation.
    -   Drill-through functionality allows users to move from overview insights to job-specific analysis.

## Dashboard Overview

*The report is organized into two pages: one for summary insights and one for focused exploration.*

### Page 1: High-Level Market View

![Data Jobs Dashboard Page 1](/images/Data_P1.png)  

This page provides a snapshot of the data job market. It highlights total job postings, median salary values, and the most common job roles, giving users an immediate understanding of overall trends and demand.

### Page 2: Job Title Drill Through

![Data Jobs Dashboard Page 2](/images/Data_P2.png)  

This page offers detailed insights for individual job titles. Users can drill through from the main view to analyze salary distributions, remote work availability, leading hiring platforms, and geographic job concentration for a selected role.

---

## Conclusion

This project demonstrates how Power BI can convert raw job posting data into meaningful, actionable insights. By combining data cleaning, visual analytics, and interactivity, the dashboard supports informed career planning and highlights key trends within the data job market.
